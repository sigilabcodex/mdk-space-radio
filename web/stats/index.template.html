Methodology and limitations

One committed source-track start event equals one play. Consecutive plays count separately.

Expected album plays are proportional to each album's number of radio-ready catalog tracks. Observed/expected values above 1 indicate overrepresentation and values below 1 indicate underrepresentation.

A nearby album repetition occurs when the same album appeared among the preceding 3, 5 or 10 plays. Current drought is elapsed UTC time since a track's last recorded play.

This is not retrospective. Missing source callbacks or catalog outages can produce gaps. “Never played” means never observed since the tracking start shown above.
